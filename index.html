TimeDropper-Ex

[Repository]

TimeDropper-Ex is a jQuery plugin for fast visual and interactive time entry.

Requires jQuery 1.7.0+ (MIT LICENSE)

Compatible Browsers:

  • Google Chrome 30+
  • Mozilla Firefox 30+
  • Microsoft Edge
  • Microsoft Internet Explorer 9+
  • Opera 20+
  • Safari 4+

No drop, two way data entry
Auto start (fade), fixed initial time
Trigger (drop), follow time
